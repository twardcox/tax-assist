## Hard Year-End Deadlines (Must Act by Dec 31)

- [ ] Make charitable contributions if itemizing
- [ ] Max out employer 401(k) contributions (payroll deadline may be earlier)
- [ ] Execute tax-loss harvesting in taxable brokerage accounts
- [ ] Pay estimated Q4 state taxes if beneficial
- [ ] Make HSA contributions if not through payroll (can contribute until April 15)
- [ ] Make Section 179 asset purchases if needed for deduction
- [ ] Make qualifying business purchases for bonus depreciation
- [ ] Execute Roth conversions if beneficial
- [ ] Make 529 contributions for state deduction (state deadline varies)
- [ ] Distribute from donor-advised fund if applicable
- [ ] Review and execute any planned family loans (must use AFR rate)
- [ ] Make annual gift tax exclusion gifts ($18,000/recipient for 2024)
- [ ] Hold Augusta Rule rentals if planning to use
- [ ] Execute installment sale closings if timing matters
- [ ] Place rental property in service if planning to claim current-year depreciation

---

## Can Be Done After Year-End But Before April 15

- [ ] IRA contributions (traditional or Roth)
- [ ] SEP-IRA contributions (with extension, up to October 15)
- [ ] Solo 401(k) contributions — employee deferrals must be made by Dec 31 for most plans
- [ ] HSA contributions
- [ ] File or extend return

## Entity / Election Actions

- [ ] S Corp election (Form 2553) — must file within 2.5 months of new tax year
- [ ] QBI aggregation elections
- [ ] Accounting method elections
- [ ] Cost segregation study — commission before year-end for current-year benefit
- [ ] Review payroll for S Corp reasonable compensation

- [ ] Property tax records
- [ ] Charitable giving receipts
- [ ] Business mileage log
- [ ] Home office measurements and photos
- [ ] Vehicle log
- [ ] Medical expense receipts
- [ ] Business receipts organized by category
- [ ] Prior year return for carryforward reference
